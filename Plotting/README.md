# Plotting Contact Difference Matrices

This directory contains scripts used to visualize **Δ contact matrices** generated by the structural contact analysis workflows.

Each plotting script reads a contact difference matrix (`delta_contact_matrix.csv`) and generates a heatmap showing gains and losses of contacts between ribosomal components.

The matrices represent the difference between a **test structure (VC-MUT)** and a **reference structure (VC-ST)**:

ΔC = C_test − C_reference

Interpretation of values:

Positive → Contact gained in mutant  
Negative → Contact lost in mutant  
Zero → No change

All plots use a diverging color scale:

Blue → contact gain  
White → no change  
Red → contact loss


------------------------------------------------------------
Input File
------------------------------------------------------------

All plotting scripts expect the same input file:

delta_contact_matrix.csv

This file is generated by the corresponding contact analysis scripts.

Matrix interpretation depends on the analysis type:

RNA–RNA contacts:
Rows → RNA residues
Columns → RNA residues

RNA–Protein contacts:
Rows → RNA residues
Columns → protein residues

Protein–Protein contacts:
Rows → ribosomal proteins
Columns → ribosomal proteins


------------------------------------------------------------
Scripts
------------------------------------------------------------

1) RNA–RNA Contact Heatmap

Script:
intra_rna_plot_v2.py

This script visualizes contact differences within the 16S rRNA.

Axes:
X → 16S rRNA nucleotide index
Y → 16S rRNA nucleotide index

Features:
• symmetric RNA contact heatmap
• mutation sites annotated
• sparse nucleotide tick marks
• high‑resolution figure output

Output file:
intra_rna_delta_contacts.png


------------------------------------------------------------

2) RNA–Protein Contact Heatmap

Script:
rna_prot_contacts_grid_FINAL_plot_mut_moved.py

This script visualizes contact differences between rRNA nucleotides and ribosomal proteins.

The script:
• aggregates RNA–protein contacts per nucleotide
• maps protein chain IDs to ribosomal protein names

Axes:
X → ribosomal proteins
Y → 16S rRNA nucleotide index

Mutation sites are annotated along the RNA axis.

Output file:
rna_protein_delta_contacts.png


------------------------------------------------------------

3) Protein–Protein Contact Heatmap

Script:
intra_protein_plot.py

This script visualizes contact differences between ribosomal proteins.

The script:
• converts chain IDs to ribosomal protein names
• orders proteins into the canonical 30S ribosomal protein order

Axes:
X → ribosomal proteins
Y → ribosomal proteins

Output file:
intra_protein_delta_contacts_q_filtered.png


------------------------------------------------------------
Running a Plot Script
------------------------------------------------------------

Example:

python intra_rna_plot_v2.py

The script will:
1. read delta_contact_matrix.csv
2. generate the heatmap
3. save the figure as a PNG file
4. display the figure

Dependencies

numpy

Install with:

pip install numpy pandas matplotlib


------------------------------------------------------------
Notes
------------------------------------------------------------

All figures are generated using Matplotlib with consistent styling:

• diverging color scale
• square heatmap layout
• high‑resolution export (600 dpi)
• gridlines for readability

These plots are designed for direct inclusion in structural biology figures and publications.
